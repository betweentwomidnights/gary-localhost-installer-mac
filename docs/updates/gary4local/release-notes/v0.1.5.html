gary4local v0.1.5

Apple Silicon localhost update with Carey XL support, foundation-1 MLX support, and Sparkle-based app updating.

Highlights

  • Adds Carey regular and XL base, sft, and turbo model-family support in the macOS app.
  • Improves Carey localhost parity with the refreshed complete, cover, and extract workflow.
  • Adds foundation-1 on the MLX path, including text generation, prompt randomization, and audio2audio.
  • Implements the in-app auto-updater in gary4local using Sparkle 2.
  • Shows the app version directly in the control center UI for release sanity.

Recommended Companion

  • gary4juce v3.0.3-mac
  • Releases page: https://github.com/betweentwomidnights/gary4juce/releases

Notes

  • This build is distributed outside the Mac App Store as a notarized DMG.
  • Sparkle signatures are included for the in-app updater flow.
  • The Stable Audio Hugging Face token still lives in Keychain; a follow-up UX pass will defer that lookup on launch.
